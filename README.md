**1.Project Overview:-**
This project is created using Power BI to study and analyze data related to PhonePe, one o leading digital payment platforms.
The main goal is to understand how people use PhonePe across different states, transaction trends over time.
________________________________________
 **2.Dataset Description:-**
 
  The dataset includes different types of information such as:
  
•	Transaction Details: Transaction amount, count, mode (UPI, recharge, bill payment, etc.).

•	Geographical Data: State, district, and city-level usage.

•	User Data: Number of registered users and app downloads.

•	Category Data: Transactions by merchant payments, money transfers, and recharges.
________________________________________
  **3.Business Objectives**
  
 The key objectives of this BI project are:
 
1.	To analyze total transactions and transaction amount across India.
	
2.	To compare PhonePe usage between different states and cities.
	
3.	To identify which transaction types (UPI, recharge, bills, etc.) are most used.
	
4.	To study growth trends of digital payments over time.
	
5.	To create an easy-to-understand Power BI dashboard showing the results.
________________________________________
 **4.Tools & Techniques**
 
•	Power BI Desktop – to design and analyze data.

•	Power Query – to clean and prepare data.

•	DAX Formulas – for calculations like total transaction amount, average per user, and growth rate.
 **•	Visuals Used:**

o	Maps (for state-wise usage)

o	Bar and line charts (for transaction trends)

o	Cards and slicers (for quick KPIs and filtering)
________________________________________
**5.Insights (Findings)**

•	Maharashtra, Karnataka, and Uttar Pradesh have the highest PhonePe usage.

•	UPI and merchant payments are the most used categories.

•	Digital payments are growing steadily month by month.

•	The average transaction value is higher in metro cities compared to rural areas.

•	States with more smartphone users and better internet connectivity show higher growth in PhonePe transactions.
